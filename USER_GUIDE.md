# 书签管理器使用指南

## 🎯 功能概览

这是一个功能完整的桌面书签管理应用，支持定时提醒功能，特别适合需要定期访问某些网站（如签到）的场景。

## 📖 详细使用说明

### 1. 添加书签

1. 点击顶部的 **"添加书签"** 按钮
2. 填写书签信息：
   - **标题**（必填）：书签的名称
   - **URL**（必填）：网站地址
   - **分类**（可选）：如"工作"、"学习"、"娱乐"等
   - **标签**（可选）：用逗号分隔，如"技术, 博客, 前端"
   - **备注**（可选）：添加说明信息

3. **设置提醒**（可选）：
   - 勾选 **"启用提醒"** 复选框
   - 选择提醒频率：
     - **每日**：每天在指定时间提醒
     - **每周**：每周特定几天提醒（可多选）
     - **自定义间隔**：每隔 N 天提醒一次
     - **一次性**：只提醒一次
   - 设置提醒时间（24小时格式）
   - 如果选择"每周"，需要选择星期几

4. 点击 **"保存"** 完成添加

### 2. 编辑书签

1. 在书签卡片上点击 **"编辑"** 按钮
2. 修改任何字段（包括提醒设置）
3. 点击 **"更新"** 保存修改

### 3. 删除书签

- 点击书签卡片上的 **"删除"** 按钮
- 确认删除操作

### 4. 打开网站

- 点击书签卡片上的 **"打开"** 按钮
- 系统会使用默认浏览器打开该网站

### 5. 搜索书签

1. 在顶部搜索框输入关键词
2. 按回车或点击 **"搜索"** 按钮
3. 支持搜索：标题、URL、分类、备注

### 6. 查看今日提醒

1. 点击顶部的 **"今日提醒"** 按钮
2. 显示所有设置了提醒的书签
3. 在提醒视图中，可以：
   - 点击 **"✓ 完成"** 标记提醒已处理
   - 点击 **"打开"** 访问网站
   - 点击 **"编辑"** 修改提醒设置

### 7. 提醒通知

当到达设置的提醒时间时：
- 系统会发送桌面通知
- 应用内会弹出确认对话框
- 可以选择标记为已完成

## 🎨 界面说明

### 书签卡片

每个书签卡片显示：
- 📌 标题和提醒图标（🔔）
- 🔗 URL 地址
- 🏷️ 分类标签（蓝色）
- 🔖 标签列表（灰色）
- 📝 备注信息
- ⏰ 提醒信息（如果设置了提醒）
- 🔘 操作按钮（打开、编辑、删除）

### 提醒设置区域

- **启用提醒**：开关提醒功能
- **提醒频率**：下拉选择
- **提醒时间**：时间选择器
- **选择星期**：星期按钮（仅每周提醒）
- **间隔天数**：数字输入（仅自定义间隔）

## 💡 使用技巧

### 1. 签到网站管理

为需要每日签到的网站设置提醒：
```
标题：某论坛签到
URL：https://example.com/signin
分类：签到
提醒：每日 09:00
```

### 2. 周期性任务

为每周需要处理的任务设置提醒：
```
标题：周报提交
URL：https://work.example.com/report
提醒：每周一、五 17:00
```

### 3. 定期检查

为需要定期查看的网站设置间隔提醒：
```
标题：域名续费检查
URL：https://domain.example.com
提醒：每 30 天 10:00
```

- 娱乐休闲：视频、音乐、游戏
- 工具网站：在线工具、开发工具

### 5. 标签使用

使用标签进行细分：

## ⚙️ 提醒工作原理

1. **后台检查**：应用每分钟检查一次是否有需要触发的提醒
2. **时间匹配**：当前时间与设置的提醒时间匹配（误差1分钟内）
3. **频率判断**：根据提醒频率判断是否应该提醒
4. **防重复**：同一天不会重复提醒
5. **通知发送**：发送系统通知和应用内提醒

## 🔧 常见问题

### Q: 提醒没有触发？
A: 确保：
- 应用正在运行
- 提醒已启用
- 时间设置正确
- 系统通知权限已开启

### Q: 如何修改提醒时间？
A: 点击"编辑"按钮，修改提醒设置后保存

### Q: 可以临时禁用提醒吗？
A: 编辑书签，取消勾选"启用提醒"

### Q: 提醒会在后台运行吗？
A: 是的，只要应用在运行，提醒服务就会在后台工作

### Q: 数据存储在哪里？
A:
- Windows: `%APPDATA%\bookmark-manager\bookmarks.db`
- macOS: `~/Library/Application Support/bookmark-manager/bookmarks.db`
- Linux: `~/.local/share/bookmark-manager/bookmarks.db`
